# Agents Configuration for Go Development

## Linting
- Command: `go vet ./...`
- Description: Runs Go's built-in vet tool to check for common errors

## Formatting
- Command: `go fmt ./...`
- Description: Formats Go source code according to standard conventions

## Testing
- Command: `go test ./...`
- Description: Runs all tests in the project

## Building
- Command: `go build ./...`
- Description: Builds the Go project

## Dependency Management
- Command: `go mod tidy`
- Description: Cleans up go.mod and go.sum files
